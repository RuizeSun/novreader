## versioning

> a.b.c-d.e+f → a > b > c > d > e (f)  
> **d:** Beta > Alpha > Snapshot

Example: `1.0.0-snapshot.1` → `1.0.0-snapshot.2` → `1.0.0-alpha.1` → `1.0.0-alpha.2` → `1.0.0-beta.1` → `1.0.0-beta.2` → `1.0.0`  
`1.0.0` → `1.0.1` → `1.0.2` → `1.1.0` → `1.2.0` → `2.0.0`
